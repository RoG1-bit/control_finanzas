# Sistema de Control de Finanzas (Desafío 20%)

Este es el repositorio base del proyecto. Aquí está la conexión, base de datos, login y dashboard.

## Instrucciones para el equipo:

1. **Clonar el proyecto:** Abran la terminal en su carpeta `htdocs` y ejecuten:
   `git clone https://github.com/RoG1-bit/control_finanzas.git`
2. **Base de Datos:**
   - Enciendan Apache y MySQL en XAMPP.
   - Vayan a `localhost/phpmyadmin` y creen una base de datos vacía llamada `control_finanzas`.
   - Vayan a la pestaña "Importar" y suban el archivo `control_finanzas.sql` que viene incluido en esta carpeta.
3. **Probar el sistema:** Vayan a `localhost/control_finanzas` e inicien sesión con:
   - **Usuario:** admin
   - **Contraseña:** password

¡Éxitos a todos con sus módulos!
